# Landing page

Task can be checked on https://landing-page-ib.netlify.app/ .

## Project Setup

```sh
npm install
```

### Compile for Development

```sh
npm run compile:scss
```

### Continuous Compile with watcher

```sh
npm run watch:scss
```

### Running the Local Server

```sh
npm install -g http-server
```

```sh
http-server ./ -o
```
